# Teapot asset

`teapot.obj` comes from the [common-3d-test-models][models] collection. The
repository describes the models as common computer graphics test models and
lists the Utah teapot source as Martin Newell's original data.

[models]: https://github.com/alecjacobson/common-3d-test-models

The renderer accepts ordinary triangular or polygonal OBJ faces and computes
smooth normals from their geometry, so this file can be opened, changed, and
re-exported from Blender.
